Browser Review

Browsers are an essential part of browsing the internet, it makes sense that you would want to be using the best browser for your needs but it doesn't always end up that way. Google Chrome has always been the go to for many people and Edge is preinstalled on every Windows machine, but what about Firefox or Opera? Each browser has their own features and I will be comparing multiple to figure out which is the best for me.

For convenience clicking on the icon of a browser will take you to the browsers website.

Privacy

How well does the browser keep your information confidential?

It's always nice knowing that you don't have to give up your privacy to browse the web, and it's even better if the browser protects you by default. Chrome isn't the best choice for privacy as it doesn't block any type of tracker by default, Edge is a bit better since it blocks third party trackers but Microsoft themselves still harvest data. Opera is supposed to be more privacy focused and does block trackers but some people don't 100% trust what the company does with data it gathers itself, Firefox also blocks trackers, and does collect some anonymized user data but it is optional.

Firefox is by far the most privacy focused of the four browsers with Opera right behind. I personally really like that Firefox also makes it easy to increase the lengths the browser will go to in an attempt to protect your data.

Convenience

How convenient is each browser?

Convenience
OS availability
Password management
Text to speech
Autoplay blocking

Availability across multiple operating systems, mobile and computer alike, alongside password management have become two features that I come to expect from a browser so it is good to see that all 4 browsers have those implemented. Text to speech and autoplay blocking aren't necessarily required for a browser, and can usually be added through extensions, but are greatly appreciated. Text to speech can be great for either if you have a reading impairment or if you would rather listen to some content, like an article. Autoplay blocking is another really nice feature, stopping the videos that automatically play on websites that can be rather annoying at times, and really makes for a calmer browsing experience in most cases.

Utility

How does each browser help the user?

Utility
Auto-fill forms
Reader mode
Extension support
In-browser screenshots

Things like auto-fill for forms can be really useful if you do things like purchase products online often, saving you time by submitting your information automatically. Extension support is also a must, no browser is going to fit each persons needs so letting the community add whatever feature they need just makes any browser so much better. Reader modes can be really helpful, providing both an environment free from clutter and usually a screen reader as well, I find it helps me stay focused when I need to read something from a site filled with ads but can also be really useful for people with reading impairments. Finally being able to take a screenshot within your browser isn't really a required feature, its really easy to take a screenshot of your desktop anyways, but can be a really nice feature if you aren't fond of the snip tool.

Conclusion

After seeing what each browser is capable of I feel like Firefox is the right browser for me. Not only is it filled with features I enjoy having, but it also respects my privacy in a very convenient way.

The browser I have been using for years now actually happens to be based on Firefox just with an even harsher stance on privacy called LibreWolf. It has a lot of the same features as Firefox, albeit not all of them, with some changes to how the browser works to force even better security.
